# NEXO Flask Store

متجر ملابس رياضية مبني باستخدام Flask و SQLite.

## التشغيل على الكمبيوتر

```bash
python app.py
```

افتح من نفس الكمبيوتر:

```text
http://127.0.0.1:5000
```

## الفتح من الهاتف على نفس شبكة Wi-Fi

1. شغل المشروع من الكمبيوتر:

```bash
python app.py
```

2. اعرف رقم IP الخاص بالكمبيوتر.

على Windows اكتب:

```bash
ipconfig
```

ابحث عن `IPv4 Address`، وسيكون شكله مثل:

```text
192.168.1.10
```

3. افتح المتصفح في الهاتف واكتب:

```text
http://192.168.1.10:5000
```

استبدل الرقم برقم IP الخاص بجهازك.

ملاحظة: يجب أن يكون الهاتف والكمبيوتر على نفس شبكة Wi-Fi. إذا لم يفتح الرابط، قد تحتاج للسماح لـ Python أو المنفذ 5000 من Windows Firewall.
